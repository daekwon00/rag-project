# Phase 5 - Step 03: 대화 공유 기능

## 목표
- 대화를 URL로 공유할 수 있는 기능 구현

## 작업 항목
- [ ] 공유 링크 생성 API (고유 share_id)
- [ ] 공유 대화 조회 페이지 (`/shared/[id]`)
- [ ] 대화 공유 버튼 UI (사이드바 또는 채팅 헤더)
- [ ] 공유 대화 읽기 전용 뷰
- [ ] 공유 링크 복사 기능

## 완료 조건
- 공유 링크를 통해 비로그인 사용자도 대화 내용 열람 가능
- 공유된 대화는 읽기 전용

## 참고
- `conversations` 테이블에 share_id 컬럼 추가 필요
- RLS 정책 조정 필요 (공유된 대화는 누구나 읽기 가능)
