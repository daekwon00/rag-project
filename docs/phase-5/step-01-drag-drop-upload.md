# Phase 5 - Step 01: 파일 드래그 앤 드롭 업로드

## 목표
- 채팅 화면에서 파일을 드래그 앤 드롭으로 업로드할 수 있도록 개선

## 작업 항목
- [ ] 드래그 앤 드롭 영역 구현 (채팅 입력 영역)
- [ ] 드롭 시 시각적 피드백 (오버레이, 하이라이트)
- [ ] 기존 파일 업로드 로직과 통합
- [ ] 다중 파일 드롭 지원
- [ ] 지원 파일 형식 검증 (PDF, TXT, MD, DOCX, PPTX)

## 완료 조건
- 파일을 채팅 영역에 드래그 앤 드롭하여 업로드 가능
- 기존 파일 선택 버튼도 동시 동작
