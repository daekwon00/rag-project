Supabase 가이드

Supabase는 오픈소스 Firebase 대안으로, PostgreSQL 기반의 Backend-as-a-Service(BaaS)입니다.

주요 기능:
1. 데이터베이스: PostgreSQL을 기반으로 한 관계형 데이터베이스를 제공합니다. SQL을 직접 사용할 수 있으며, Row Level Security(RLS)로 데이터 접근을 제어합니다.
2. 인증(Auth): 이메일/비밀번호, OAuth(Google, GitHub 등), Magic Link 등 다양한 인증 방식을 지원합니다.
3. 스토리지: 파일 업로드 및 관리 기능을 제공합니다. 이미지, 동영상 등 대용량 파일을 저장할 수 있습니다.
4. 실시간(Realtime): PostgreSQL의 변경사항을 실시간으로 클라이언트에 전송합니다. 채팅, 알림 등에 활용됩니다.
5. Edge Functions: Deno 기반의 서버리스 함수를 실행할 수 있습니다.

pgvector 확장:
Supabase는 pgvector 확장을 지원하여 벡터 데이터를 저장하고 유사도 검색을 수행할 수 있습니다.
- vector 타입: 고차원 벡터를 저장하는 PostgreSQL 데이터 타입입니다.
- <=> 연산자: 코사인 거리(cosine distance)를 계산합니다. 값이 작을수록 유사합니다.
- 인덱스: ivfflat 또는 hnsw 인덱스로 검색 성능을 최적화할 수 있습니다.

무료 티어:
- 500MB 데이터베이스 저장소
- 1GB 파일 스토리지
- 50,000 월간 활성 사용자
- 무제한 API 요청

Supabase JS SDK:
npm install @supabase/supabase-js
const supabase = createClient(url, key);
데이터 조회: supabase.from('table').select('*')
데이터 삽입: supabase.from('table').insert({ ... })
RPC 호출: supabase.rpc('function_name', { params })
