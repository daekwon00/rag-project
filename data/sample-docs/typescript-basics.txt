TypeScript 기초 가이드

TypeScript는 JavaScript에 정적 타입 시스템을 추가한 프로그래밍 언어입니다. Microsoft에서 개발했으며, JavaScript의 상위 집합(superset)입니다.

기본 타입:
- string: 문자열 타입. 예시: let name: string = "홍길동";
- number: 숫자 타입. 정수와 소수를 모두 포함합니다. 예시: let age: number = 30;
- boolean: 참/거짓 타입. 예시: let isActive: boolean = true;
- array: 배열 타입. 예시: let items: string[] = ["a", "b"];
- object: 객체 타입. interface나 type으로 구조를 정의합니다.

인터페이스(Interface):
객체의 구조를 정의하는 방법입니다. 클래스가 구현해야 할 메서드와 속성을 명시합니다.
예시:
interface User {
  id: number;
  name: string;
  email?: string;  // optional 속성
}

제네릭(Generics):
타입을 매개변수로 받아 재사용 가능한 컴포넌트를 만드는 기능입니다.
예시: function identity<T>(arg: T): T { return arg; }

유니온 타입(Union Types):
여러 타입 중 하나일 수 있는 값을 표현합니다.
예시: let value: string | number = "hello";

TypeScript의 장점:
1. 컴파일 시점에 타입 오류를 발견할 수 있어 런타임 에러가 줄어듭니다.
2. IDE의 자동완성과 리팩토링 지원이 향상됩니다.
3. 코드 문서화 효과가 있어 팀 협업에 유리합니다.
4. JavaScript 생태계와 100% 호환됩니다.
